	node.js

	js
front-end:
	html5,css,js
	paper.js
	jquery
	boostrap (TBD)
======================================
=== How to start
1. install github client (windows)
2. install nodejs which contains npm
3. clone projects to local
4. start gitshell command window
5. type "npm install ." to install modules
6. type "npm run start" to start server

Refer to doc “Project Information.doc”
